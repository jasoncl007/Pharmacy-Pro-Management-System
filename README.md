A complete pharmacy/medicine management system built with Python. **Import thousands of medicines from Excel automatically in one click!** Manage stock, sell by box or piece, generate bills, track inventory, and get low stock alerts.

## 📋 Excel File Format Required

Your Excel file must have these columns in this order:

| Column | Data | Example |
|--------|------|---------|
| 1 | Medicine Name | Napa |
| 2 | Company | Square Pharmaceuticals |
| 3 | Strength | 500 mg |
| 4 | Dosage Form | Tablet |
| 5 | Slug | napa-500mg-tablet |
| 6 | Unit Price | 5.00 |

## 🚀 How to Use

### First Time Setup (30 minutes - ONE TIME ONLY)

1. Place your `medicine.xlsx` file in the correct folder
2. Run the program
3. Click **"YES"** for auto-import
4. Wait 30 minutes (coffee break ☕)
5. Done! Your medicines are ready!
